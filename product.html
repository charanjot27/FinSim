About the product

A financial metaverse for the rest of us.

FinSim is a browser-based open-world RPG that teaches real finance through play. Six districts, twenty-seven NPC dialogues, fifteen historic crashes to replay, a real-time behavioural-bias detector, and a Firestore-backed leaderboard where you stack up against bots and live players. No real money at risk. No fake lessons.

Six districts, six curricula

Scam Slum

Spot the pitch. Walk away from the guaranteed doubler.

Wall Street

Stop-losses, position sizing, the 1% rule.

Dalal Street

Indian microstructure, volume before price, gap risk.

Crypto Cove

24/7 volatility, self-custody, rug-pull verification.

Quant Quarter

Sharpe, factor models, the backtest-honesty check.

Vegas Vice

Negative expectancy made viscerally obvious.

What makes it different

Real data, real news

Live prices (Finnhub / Alpha Vantage), ticker-tagged headlines (Marketaux), macro (FRED), gas (Etherscan).

Bias detector

Overtrading, revenge trades, sunk-cost — caught in-session, not post-mortem.

Echo Mode

Replay 1987, 2008, LUNA, FTX. You decide; game reveals what actually happened.

Guided tour

Mira, a retired floor trader, walks you through every district stop by stop.

Real leaderboard

Firestore-backed. See live players + seeded bots on one board.

Offline-first

Every API has a curated fallback. Learning never breaks when you’re offline.

Tech & scale

Phaser 3 + TypeScript

75 modules, strict types, Vite build, ~2.2 MB bundle.

Firebase

Google auth + Firestore for cloud save & multiplayer leaderboard.

DOM overlays

Hybrid: Phaser for the world, DOM for every HUD and terminal — maximum UI flexibility.

Singleton + pub/sub

No state library. Every system notifies subscribers. Simple, debuggable, fast.
